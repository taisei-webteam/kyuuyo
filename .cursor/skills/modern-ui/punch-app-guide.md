# iPad PWA 打刻アプリ UI ガイド

## 画面構成

打刻アプリ (`punch-app/`) は iPad 横置き・縦置き両対応のシンプルな単画面アプリ。

### レイアウト

1. **ヘッダー**: アプリ名 + 現在日時 (大きなデジタル時計) + オフラインバッジ
2. **フィルタバー**: 「全員 / 社員 / 役員 / パート」の切替タブ
3. **カードグリッド**: 従業員カード (4列 / iPad横置き時)
4. **確認ダイアログ**: 名前タップ時のモーダル

### デザイン原則

- タッチ操作に最適化: カードは最低 80x80px、タップ領域は十分な大きさ
- 高コントラスト: 打刻端末として離れた位置からも読める文字サイズ
- 状態の視覚的区別: 未打刻(白)、出勤中(緑)、退勤済み(グレー)
- 確認ダイアログ: 誤タップ防止のため必ず1ステップ確認を挟む

### カラー

- ヘッダー: `--color-primary` (#2563eb, ブルー)
- 出勤中カード: `--color-success-light` (薄緑)
- 退勤済みカード: `--color-done-light` (薄グレー) + opacity 0.7
